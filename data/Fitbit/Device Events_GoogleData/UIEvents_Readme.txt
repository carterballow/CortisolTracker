UIEvents.json

This file contains events logged by your Fitbit device's UI system.

Each entry is a JSON object that includes one of the following fields:
   - auto_wake
   - button_lock_toggle
   - hr_notifications
   - stress_tile
   - oobe
   - tiles_config
   - popup_hfp
   - gmaps_app
   - aspen_config
   - music_app
   - exapp_settings
   - exapp_session
   - auto_detect
   - hr_zones_set
   - data_provider
   - ar_settings
   - fbui_status
