ChargerEvents.json

This is the list of events logged by your Fitbit device's charger system.
Each entry is a JSON object that includes a mandatory "event" field, plus exactly one of the following:
   - charger_state
   - bq25120_state
   - max14676_state
   - charge_cycles
   - bq25150_state
   - regdump
   - bq25120_state_fault
   - battery_charge_dcr
   - vbus_short_detection
   - battery_accounting
   - bq25190_state
