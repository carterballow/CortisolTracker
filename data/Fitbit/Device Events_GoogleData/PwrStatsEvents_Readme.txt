PwrStatsEvents.json

This file contains events logged by your Fitbit device's power monitoring system.

Each entry is a JSON object that includes a mandatory "no_of_events", "duration_ms", and "app_name" fields, plus exactly one of the following:
   - mcu
   - display
   - ui
   - fwup
   - touch
   - btn
   - wrist
   - ble
   - hrm
   - vibe
   - flash
   - clock_face
   - fs
   - gnss
   - aod
   - charger
   - gestures
