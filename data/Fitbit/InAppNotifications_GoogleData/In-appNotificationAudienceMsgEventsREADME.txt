In-app Notification Audience Message Events Data Export

In-app Notification Audience Message Events Export provides a way to export your in-app notification audience message events data.
The data is exported in CSV format.

Files Included:
----------

in-app-notification-audience-messages-events.csv - The CSV file containing the In-app Notification Audience Messages Events.

This file contains the following columns:
  id                              - The unique identifier of the In-app Notification Audience Message Event.
  template_id                     - The template id of the In-app Notification Audience Message.
  title                           - The title of the In-app Notification Audience Message.
  body                            - The body of the In-app Notification Audience Message.
  priority                        - The priority of the In-app Notification Audience Message.
  destination                     - The destination id of the In-app Notification Audience Message.
  creation_time                   - The creation time(Unix timestamp) of the event of the In-app Notification Audience Message Event.
  event_type                      - The type of the In-app Notification Audience Message Event.
